# Buffet Biruta Park — Site + Orçamento Online

## Publicação no Render

Build Command:
```
yarn
```

Start Command:
```
yarn start
```

## Acessos

Painel administrativo:
```
/admin
```

Login inicial:
```
17717592000160
```

Senha inicial:
```
Biruta@2026
```

Dentro do painel, acesse **Login e senha** para trocar.

## Tamanho recomendado dos banners

- Desktop/topo do site: **1920 x 760 px**
- Área segura central: manter textos e elementos principais no centro
- Versão vertical para redes/stories, se quiser reaproveitar: **1080 x 1350 px**

## Manutenção

- Pacotes: `/admin/pacotes`
- Banners, promoções, depoimentos, logo e favicon: `/admin/site`
- Solicitações de orçamento: `/admin/leads`

O sistema não calcula valores automaticamente. Ele coleta os dados do cliente, pacote de interesse e envia a solicitação pelo WhatsApp.
